# StudyLab Free v6 complete

無料会員向けの完成版です。バックエンドなしで使える範囲に絞っています。

実装済み
- ダッシュボード
- 自習室（カメラ / タイマー / 休憩 / メモ / セッション保存）
- タスク管理（追加 / 状態変更 / 削除）
- 学習計画（追加 / 一覧 / 曜日表示 / 削除）
- 学習記録（保存データから集計）
- 目標設定（追加 / 加算 / 削除）
- 学習のコツ（モニター表示 / カテゴリ切替 / ランダム）
- バックアップ保存 / 復元 / 全削除

保存方式
- localStorage（この端末・このブラウザ内）
- バックアップJSONを書き出して復元可能
