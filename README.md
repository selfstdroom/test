# オンライン自習室 v14 one-screen app

## 方針
- 1画面完結型
- 下部メニューで機能ごとに切替
- 画面全体は常に同じページ内で完結
- 見た目をアプリ寄りに整理

## 変えたところ
- 上部は常時表示のダッシュボード
- 下部メニューを 5分類に整理
  - 操作
  - タスク
  - ルーム
  - 記録
  - 設定
- 画面遷移ではなく、同一画面内のセクション切替に変更
- v13 の主要機能は維持

## ローカル確認
```bash
python -m http.server 8000
```

開くURL:
```text
http://localhost:8000
```
