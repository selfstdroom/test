# オンライン自習室 v15 no-scroll zoomlike

## 方針
- スマホでスクロールなし
- Zoom っぽく「上は固定情報 / 下はモード切替」
- その場で必要な機能だけ見せる
- 設定や次回目標はモーダルに退避

## 画面構成
- 上部：常時表示ダッシュボード
- 下部：4タブ
  - 操作
  - タスク
  - ルーム
  - 記録
- 設定：右上ボタンからモーダル
- 次回目標：退室後レポート内で入力

## ローカル確認
```bash
python -m http.server 8000
```

開くURL:
```text
http://localhost:8000
```
