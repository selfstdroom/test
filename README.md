# オンライン自習室 v17 prejoin flow

- 入室前はトップページとして動作
- 入室前は監視感を出さない
- 目標時間 / 休憩 / 講師名 / 次回目標 / タスク登録をしてから入室
- 入室後に集中ルームへ切り替え
- 使う流れに沿って画面を2段階に整理

## ローカル確認
```bash
python -m http.server 8000
```

開くURL:
```text
http://localhost:8000
```
